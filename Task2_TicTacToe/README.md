# Tic-Tac-Toe AI using Minimax

## Description
This project is a human vs AI Tic-Tac-Toe game developed using Python.  
The AI player uses the Minimax algorithm to make optimal moves and is designed to be unbeatable.

## Features
- Human vs AI gameplay
- Unbeatable AI using Minimax
- Console-based interface

## Technologies Used
- Python 3

## How to Run
python tic_tac_toe.py
