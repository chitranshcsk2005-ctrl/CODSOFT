# Recommendation System

## Description
This project is a content-based recommendation system that suggests movies based on genre similarity.  
It uses cosine similarity to find movies that are most similar to the user’s choice.

## Features
- Content-based filtering
- Movie recommendations
- Simple console interface

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
python recommendation.py
